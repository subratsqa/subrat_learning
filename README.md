# Subrat Learning

This repository contains comprehensive JavaScript learning exercises and projects. It serves as a place to practice, experiment, and grow technical skills through hands-on coding.

## Projects and Exercises

### Arrays (`array/`)
- `array-1.js` - Basic array operations and methods
- `array_asn_des.js` - Array ascending and descending sorting
- `array_sliceing.js` - Array slicing operations
- `array_spliceing.js` - Array splicing operations

### For Loop (`for_loop/`)
- `for_loop1.js` - Basic for loop exercises and implementations
- `for_Each_array.js` - forEach array iteration
- `for_in.js` - for...in loop examples
- `itearate_array_for_loop.js` - Array iteration with for loops

### Functions (`Functions/`)
- `_function_iife.js` - Immediately Invoked Function Expressions
- `arrow_function.js` - Arrow function syntax and usage
- `clouser.js` - Closure concepts and examples
- `function_as_argument.js` - Functions as arguments (callbacks)
- `function_default.js` - Default function parameters
- `rest_of_the_params.js` - Rest parameters in functions

### String Operations (`String_Operation/`)
- `05_JS_Basics.js` - JavaScript string basics
- `string_porperties.js` - String properties and methods
- `Operator/` - String operator examples

### Objects (`Objects/`)
- Object creation and manipulation exercises

### Callbacks (`Callback/`)
- Callback function patterns and implementations

### Switch Case (`switch_case.js/`)
- `switch_case1.js` - Switch case statement examples and problems

### While Loop (`while_loop/`)
- `while_loop1.js` - While loop exercises and implementations

### setTimeout (`setTimeout/`)
- Asynchronous timeout and timing exercises

### 22nd May Task (`22thMay_task/`)
- `tringle_check.js` - Triangle validation and classification exercises

### Playwright Learning (`playwright_learning/`)
- `example.spec.ts` - Playwright automated testing examples

### Promises (`Promise/`)
- `promise_api_call.js` - Promise-based API call examples
- `promise_daycheck_check.js` - Promise patterns for day validation
- `promise_prg_1.js` - Basic promise programming exercises
- `promise_with2then.js` - Promise chaining with multiple .then() calls

## Learning Goals

- Master fundamental JavaScript concepts
- Practice control flow structures (loops, conditionals)
- Understand functions, closures, and callbacks
- Work with arrays, strings, and objects
- Build problem-solving skills through coding exercises
- Develop clean code practices
- Learn asynchronous programming patterns

## Getting Started

1. Clone the repository
2. Navigate to the project directory
3. Run JavaScript files using Node.js: `node filename.js`
4. For Playwright tests: `npx playwright test`

## Requirements

- Node.js (v14 or higher)
- npm packages (see package.json)
