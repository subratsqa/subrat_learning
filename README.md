# Subrat Learning Repository

This repository contains basic JavaScript programs, scripts, and exercises for learning and practice.

## Contents
- `JS_Basics.js`: Basic JavaScript program to add two numbers
- `String_Operation/test.js`: Example for string operations

## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/subratsqa/subrat_learning.git
   ```
2. Open and run JavaScript files in your preferred editor or with Node.js:
   ```bash
   node JS_Basics.js
   ```
